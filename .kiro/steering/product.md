# Product Overview

Sakura AI is an AI-driven full-stack test automation platform that transforms testing workflows from prototype to execution.

## Core Product Features

**AI Test Case Generator**: Converts Axure prototypes into comprehensive test cases in 10-30 minutes (vs 2-3 days manually). Achieves 85-95% coverage with intelligent boundary value and risk scenario identification.

**Natural Language Test Execution**: Execute browser automation using plain Chinese descriptions instead of writing Selenium/Playwright code. Features AI-powered element detection and real-time monitoring with screenshots.

**RAG Knowledge Base**: Integrates team testing experience through vector search (Qdrant + Aliyun Embedding). Enhances AI generation with business rules, test patterns, historical issues, and risk scenarios.

**Full-Stack Test Management**: Multi-project version control, AI bulk updates, comprehensive statistics, and modern UI with multiple view modes (card, table, kanban, timeline).

## Target Users

- QA Engineers and Test Teams
- Product Managers working with Axure prototypes  
- Development teams needing automated testing
- Organizations seeking to reduce manual testing overhead

## Key Value Propositions

- **Efficiency**: 20-50x faster test case creation
- **Coverage**: 85-95% functional coverage vs 60-70% manual
- **Accessibility**: No coding required for test execution
- **Intelligence**: AI-powered with team knowledge integration
