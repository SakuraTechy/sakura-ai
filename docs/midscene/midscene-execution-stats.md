## 附录：日志解析脚本

如需自动解析 Midscene 日志并生成统计报告，可使用以下脚本：

```bash
# 查看缓存命中情况
grep "cache" ./midscene_run/log/*.log

# 查看Token统计
grep -E "token|cost" ./midscene_run/log/*.log

# 查看AI调用耗时
grep "AI.*耗时" ./logs/debug-execution.log
```

或使用我们提供的日志解析服务：
- 文件: `server/services/midsceneLogParser.ts`
- API: `GET /api/midscene-report/:runId/stats`
