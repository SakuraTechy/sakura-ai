## 🔄 数据流图

### 1. 测试用例生成流程

```
用户选择项目
    │
    ▼
┌─────────────────────┐
│ 前端：验证项目配置   │
│ validateProjectConfig│
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ API：获取配置验证结果│
│ GET /validate-config│
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ 服务：查询默认配置   │
│ - 默认账号          │
│ - 默认服务器        │
│ - 默认数据库        │
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ 数据库：查询配置表   │
│ WHERE is_default=true│
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ 返回验证结果        │
│ { valid, missing,   │
│   warnings }        │
└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ 前端：显示验证结果   │
│ - 配置完整：绿色提示 │
│ - 配置不完整：警告   │
└─────────────────────┘
```

```
│ 服务：查询用例配置   │

└──────────┬──────────┘
           │
           ▼
┌─────────────────────┐
│ 数据库：查询        │

└─────────────────────┘
```

## 🎯 配置优先级

```
┌─────────────────────────────────────────┐
│          配置优先级（从高到低）          │
├─────────────────────────────────────────┤
│                                         │
│  1️⃣ 测试用例级别配置（未实现）          │
│     └─> 用例特定的账号/服务器/数据库     │
│                                         │
│  2️⃣ 项目默认配置（当前实现）            │
│     └─> 项目管理模块配置的默认值         │
│                                         │
│  3️⃣ 系统默认配置（降级方案）            │
│     └─> 配置缺失时的默认值               │
│                                         │
└─────────────────────────────────────────┘
```

## 📊 配置验证流程

```
┌─────────────────────────────────────────┐
│           配置验证流程图                 │
└─────────────────────────────────────────┘

开始验证
    │
    ▼
┌─────────────────────┐
│ 查询默认账号        │
│ is_default=true     │
│ status='active'     │
└──────────┬──────────┘
           │
           ├─> 存在 ──┐
           │          │
           └─> 不存在 ─┼─> missing.push('默认测试账号')
                      │
                      ▼
           ┌─────────────────────┐
           │ 查询默认服务器       │
           │ is_default=true     │
           │ status='active'     │
           └──────────┬──────────┘
                      │
                      ├─> 存在 ──┐
                      │          │
                      └─> 不存在 ─┼─> missing.push('默认测试服务器')
                                 │
                                 ▼
                      ┌─────────────────────┐
                      │ 查询默认数据库       │
                      │ is_default=true     │
                      │ status='active'     │
                      └──────────┬──────────┘
                                 │
                                 ├─> 存在 ──┐
                                 │          │
                                 └─> 不存在 ─┼─> warnings.push('未配置默认数据库')
                                            │
                                            ▼
                                 ┌─────────────────────┐
                                 │ 返回验证结果        │
                                 │ valid = missing为空 │
                                 │ missing = []        │
                                 │ warnings = []       │
                                 └─────────────────────┘
```
